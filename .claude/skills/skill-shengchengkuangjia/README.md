# Skill 生成框架

> 一个通用的、可复用的 Skill 生成系统，能够快速创建专业级别的 Claude Skill

---

## 🎯 框架概述

**Skill 生成框架** 是一套系统化的方法论和工具，用于快速生成高质量的 Claude Skills。

通过这个框架，你可以在 **约 1-2 小时内** 为任何角色或领域创建一个完整的、生产级别的 Claude Skill。

### 核心理念

```
用户输入 (角色/领域信息)
    ↓
交互式问卷 (系统化收集信息)
    ↓
框架模板 (结构化处理)
    ↓
自动生成 (SKILL.md + scripts + reference)
    ↓
质量检查 (验证完整性)
    ↓
完成的 Skill (立即可用)
```

## 📋 5维度问卷说明

框架的核心是 **5维度交互式问卷**，用于系统化地收集信息：

### 1️⃣ 角色定义 (Role Definition)
```
问题:
  • 这个角色的官方职称？
  • 主要工作职责是什么？(5-7个)
  • 核心技能要求？(5-7个)
  • 日常工作的主要挑战？
```

### 2️⃣ 知识体系 (Knowledge Domain)
```
问题:
  • 这个角色需要掌握的关键概念？(10-15个)
  • 最佳实践和常见模式？(5-10个)
  • 行业标准和规范？
  • 关键工具和方法论？
```

### 3️⃣ 工作场景 (Use Cases)
```
问题:
  • 这个角色最常见的工作场景？(5-7个)
  • 每个场景的完整工作流程？
  • 关键决策点和注意事项？
  • 常见的失败案例？
```

### 4️⃣ 工具和方法 (Tools & Methods)
```
问题:
  • 常用的分析工具？
  • 常用的方法论？
  • 模板和框架？
  • 自动化工具？
```

### 5️⃣ 特殊需求 (Special Requirements)
```
问题:
  • 行业特定的合规要求？
  • 市场特定的考虑因素？
  • 公司特定的流程？
  • 其他特殊需求？
```
