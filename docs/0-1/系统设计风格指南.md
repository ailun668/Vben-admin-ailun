## 1. 设计理念

本设计风格旨在打造一个现代化、富有吸引力且用户友好的界面。核心采用 **玻璃拟物化 (Glassmorphism)** 风格，通过半透明、模糊、阴影和渐变背景的结合，创造出富有深度和层次感的视觉体验。

## 3. 核心风格：玻璃拟物化 (Glassmorphism)

这是本次设计的核心，所有容器（如卡片、头部、导航栏）都应遵循此风格。

### 3.1 CSS 实现

```css
.glass-effect {
  /* 核心三要素 */
  background: rgba(255, 255, 255, 0.85); /* 半透明背景 */
  backdrop-filter: blur(10px);          /* 背景模糊 */
  border: 1px solid rgba(255, 255, 255, 0.2); /* 微光边框 */

  /* 辅助样式 */
  border-radius: 1rem; /* 16px 圆角 */
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.1); /* 柔和阴影 */
}
```

### 3.2 Tailwind CSS 应用

虽然 Tailwind 默认不完全支持 `backdrop-filter`，但可以通过 JIT 模式或在 `tailwind.config.js` 中扩展。为了简单起见，我们可以在全局 CSS 中定义 `.glass-effect` 类，并在需要的地方应用它。

## 4. 布局与间距 (Layout & Spacing)

- **圆角 (Border Radius)**:
  - **主要容器**: `rounded-2xl` (1rem, 16px)
  - **按钮/输入框**: `rounded-xl` (0.75rem, 12px)
  - **标签/徽章**: `rounded-full`
- **阴影 (Box Shadow)**:
  - **主要容器**: `shadow-xl`
  - **悬浮效果**: `shadow-2xl`
- **间距 (Spacing)**:
  - **容器内边距**: `p-6` (1.5rem, 24px) or `p-8` (2rem, 32px)
  - **元素间距**: `space-y-4` (1rem, 16px) or `gap-4`

---

## 5. 字体排版 (Typography)

- **字体**: `Inter` (可以通过 `@import` 引入)
- **标题**:
  - `h1`: `text-3xl font-bold text-gray-800`
  - `h2`: `text-2xl font-bold text-gray-800`
  - `h3`: `text-lg font-bold text-gray-800`
- **正文**: `text-base text-gray-600` (1rem, 16px)
- **辅助文本**: `text-sm text-gray-500` (0.875rem, 14px)

## 6. 组件风格指南

### 6.1 按钮 (Buttons)

- **主按钮**:
  - `bg-indigo-600 text-white font-semibold rounded-xl py-3 px-6`
  - `hover:bg-indigo-700`
- **次要按钮**:
  - `bg-white text-gray-700 font-semibold rounded-xl py-3 px-6 border border-gray-200`
  - `hover:bg-gray-50`

### 6.2 输入框 (Inputs)

- `w-full px-4 py-3 border border-gray-300 rounded-xl`
- `focus:ring-2 focus:ring-indigo-500 focus:border-transparent`

### 6.3 标签页 (Tabs) - 针对新功能

建议将多标签页导航栏改造为符合玻璃拟物化风格。

**`MultiTabs.vue` 样式建议**:

```css
.multi-tabs-container {
  /* 应用玻璃效果 */
  background: rgba(255, 255, 255, 0.85);
  backdrop-filter: blur(10px);
  padding: 8px 16px 0;
  border-bottom: 1px solid rgba(0, 0, 0, 0.05);
}

/* 移除 antd 默认背景和边框 */
:deep(.ant-tabs) {
  background: transparent;
}
:deep(.ant-tabs-nav) {
  margin-bottom: 0 !important;
}
:deep(.ant-tabs-nav::before) {
  border-bottom: none !important;
}

/* 标签页样式 */
:deep(.ant-tabs-tab) {
  background: rgba(255, 255, 255, 0.5) !important;
  border: 1px solid transparent !important;
  border-bottom: 1px solid rgba(0, 0, 0, 0.05) !important;
  border-radius: 8px 8px 0 0 !important;
  transition: all 0.3s ease;
}

/* 激活的标签页 */
:deep(.ant-tabs-tab-active) {
  background: #fff !important;
  border-color: rgba(0, 0, 0, 0.05) !important;
  border-bottom-color: #fff !important;
  font-weight: 600;
  color: #4F46E5 !important;
}
```

## 7. 开发实施建议 (@开发)

1.  **全局样式**: 在主样式文件 (如 `src/style.css`) 中引入 `Inter` 字体，并定义全局的渐变背景。
2.  **Tailwind 配置**: 如果 `backdrop-filter` 等工具类不可用，需要在 `tailwind.config.js` 中进行扩展。
3.  **组件改造**:
    *   首先从 `MultiTabs.vue` 开始，应用上述 CSS 覆盖样式，使其符合新的设计风格。
    *   逐步改造 `App.vue` 的 `a-layout-header` 等核心布局组件，应用 `.glass-effect` 风格。
    *   调整 `ant-design-vue` 的主题变量 (在 `vite.config.ts` 或 Less/CSS 变量中)，将主色调更新为 `--indigo-600` (`#4F46E5`)。
4.  **图标**: 保持使用 `@ant-design/icons-vue`，但可以考虑在关键操作中使用 SVG 或双色图标来增强视觉效果。
